# 2.错误代码

## 2.1.法一：

```C++
class Solution
{
public:
    int searchInsert(vector<int> &nums, int target)
    {
        int left = -1, right = nums.size(), pos; // 有问题
        while (left <= right)                    // 有问题
        {
            pos = left + ((right - left) / 2);
            if (nums[pos] == target)
                return pos;
            else if (nums[pos] < target)
                left = pos + 1; // 问题！
            else
                right = pos - 1; // 问题！
        }
        return pos + 1; // 有问题
    }
};
```

# 3.AC代码

## 3.1.1法一:

```C++
class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int left = -1, right = nums.size();
        while (left + 1 != right)
        {
            int mid = left + (right - left) / 2;
            if (nums[mid] >= target)
                right = mid;
            else
                left = mid;
        }
        return left + 1;
    }
};
```

## 3.1.2.复杂度分析

**时间复杂度：** `O(logN)`

**空间复杂度：** `O(logN)`
